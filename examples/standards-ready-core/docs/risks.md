# Risks and Mitigations

| Risk                                        | Mitigation                                                                    |
| ------------------------------------------- | ----------------------------------------------------------------------------- |
| Example evidence mistaken for certification | Every safety/security artifact states that no certification is claimed.       |
| Tool availability differs by host           | `af core standards doctor` reports local tool availability and install hints. |
| Backpressure misuse                         | Ready/valid semantics are documented in `docs/spec.md`.                       |
